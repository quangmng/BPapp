# BPapp
Blood Pressure recording app (w/ SQLite under the hood), for iOS 6 devices. Part of 30-day Objective-C challenge, available at [my blog page](https://qmng.notion.site/objective-c-challenge).

(P.s.: This branch references the tutorial of [@profgustin on YouTube](https://www.youtube.com/playlist?list=PLLk083vNjCXYTT_oLTDlGq2gelsILnLIT). Some SQLite elements can be further secured, and avoiding bad practices such as the use of [stringWithFormat](https://github.com/quangmng/BPapp/tree/no-stringWithFormat) which can be vulnerable to SQL Injection attacks ([@robertmryan in comments](https://youtu.be/uxoeZ-1yCMc)). More updates will be made on the other branch.)
